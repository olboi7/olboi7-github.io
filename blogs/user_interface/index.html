User interface

First version of the user interface

The main window

In this window you can access command graphics or the system map.

The command window

If you want to change cycle time or open/close one valve to check if all works. Whith that it’s easier to see if there is clogged foggers I don’t have to restart my arduino or wait for another cycle.

The map window

On this window you have all the sensors real time data. Under the sun it’s the temperature, under the cloud it’s the humidity, on the pressure tank it has the pressure in the water tank you have the amount of ppm and the ph. Also, the red rectangles represent a water sensor. Next to them you have the status. If the status is ok there is no leak. If some water goes on a sensor the status will change to leak. With that it’s easier to trouble shooting if a leak appends. You can also click on the sun, the cloud, the pressure tank dial and the little rectangle above the water tank. Those buttons access to their respective graphics. It’s also possible to readjust the ph and ppm command.

the pressure graphic

That graphic is to track the pressure. If the pressure upper peak start to drop that means there is a problem. It could be a leak or it could be a pump issue. If it’s a motor issue I will plan to change the pump at the end of the current harvest (preventive maintenance).
